## 📌 Visão Geral do Projeto
O projeto consiste em uma aplicação mobile desenvolvida utilizando o React Native com Expo, com o objetivo de exibir informações básicas relacionadas a dados acadêmicos, como:
Nome, Curso, Disciplina, Descrição. A aplicação possui uma estrutura simples e voltada para aprendizado ou prototipagem inicial.

## 🛠️ Tecnologias Utilizadas
### 📱 React Native
Framework utilizado para desenvolvimento mobile.
Permite criar aplicações nativas usando JavaScript.

### ⚡ Expo
Ferramenta que facilita a criação, execução e testes do app.
Remove a necessidade de configurações complexas de ambiente nativo.

## 📌 Conclusão

O projeto é uma aplicação mobile simples, com foco educacional, que demonstra o uso de: Componentes React Native, Props, Configuração com Expo.
Ele serve como uma boa base para evoluir para aplicações mais complexas, adicionando interatividade, persistência de dados e melhor experiência do usuário.
